# Тема проекта "Карточки для запоминания"

## О проекте
Мини-веб-приложение для создания наборов карточек и заучивания терминов. Реализован поиск и режим тренировки с подсчетом процента правильных ответов.

## Установка и запуск
1. Клонировать репозиторий: `git clone https://github.com/Update-lofi/SR-TRPO.git`
2. Установить зависимости: `pip install Flask Flask-SQLAlchemy pytest`
3. Запустить приложение: `python app.py`
Приложение будет доступно по адресу `http://127.0.0.1:5000`

## Запуск тестов
Для запуска автоматических тестов выполните в корне проекта:
`pytest`
