# Требования к проекту «Карточки для запоминания»

## 1. Проблема
При изучении иностранных языков или сложных технических терминов пользователям трудно систематизировать информацию и отслеживать свой прогресс. Приложение решает боль неэффективного заучивания путем создания структурированных наборов карточек с режимом проверки знаний.

## 2. Целевая аудитория
Студенты, школьники, люди, изучающие иностранные языки, а также специалисты, которым необходимо запоминать большие объемы терминологии.

## 3. Функциональные требования
1. Создание, просмотр, редактирование и удаление наборов карточек.
2. Добавление, редактирование и удаление карточек внутри набора.
3. Режим тренировки: показ вопроса, ввод ответа пользователем, проверка правильности.
4. Ведение статистики: расчет процента правильных ответов.
5. Поиск и фильтрация наборов по названию.
